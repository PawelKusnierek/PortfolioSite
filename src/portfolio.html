My portfolio & projects

  • Letter Typing Game - (Note: best results on a PC/laptop!) a little game I created when I realised that most websites attempting to teach touch typing do so by having you type full words. By typing individual letters you are forced into learning the proper finger placement technique rather than typing words out of sheer memory. My personal record is 115! See if you can beat it.
    I am also currently working on adding a MySQL database allowing me to track highscores of users.
  • Sketch Book - my attempt at a sketch book built using JavaScript. Make sure to check out the different grid sizes!
  • JavaScript Drum Kit - (Note: needs speakers/headphones!) a rather jokey project where I replaced a few Drum Kit sounds with me saying a few web related words. But you can still play the Drums on the bottom row!
    I am currently working on a similar project but using a piano instead of a drum kit. Stay tuned!
  • Tic Tac Toe - a 2-player Tic-Tac-Toe game. For when you don't have pen and paper on hand!
  • Kusnierek Recipes - this is my first ever website, currently rather outdated and written using subpar practices and styling. I like to look at it in order to remind myself how far I've come... and it is still rather useful when I cannot decide what I want to eat for dinner!
    I am also currently working on rewriting the entire site using WordPress. Should be available soon!
