Resources

In this section I would like to share some of the most useful resources I've come across over the years learning web development. Some of these are tutorials, other are just useful resources that help with building websites. I highly recommend them & hope that someone will find them as useful as I did!

Without further ado:

  • Grid Layout Designer - a website allowing you to sketch out your CSS Grid layout and even provide you with the necessary code snippet, really speeds up setting up of CSS Grid!
  • Font Awesome - a website containing a lot of free icons and logos. There's a premium version that contains more variations but personally I was always satisfied with the free ones!
  • uiGradients - the name of the website pretty much sums it up. It's a gradient generator, you can select the mixture of colors you're looking for and pick from a huge collection! They even provide fallback code for older browser versions.
  • Tania Rascia Blog - the best personal blog/portfolio/tutorial site I have come across. Lovely site to navigate and contains a ton of useful tutorials are articles.
  • CodeWars - a site to practice your coding and problem solving skills. If you're feeling rusty with any particular language this is the site to check out.
  • Heroku - finally a shoutout to Heroku who provide a free hosting and deployment service. Most of my sites are deployed on Heroku for testing before I buy a domain name and release the site.
